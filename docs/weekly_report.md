
# The RSC Weekly Report

The phenotype collection currently covers many of the [data extraction processes](weekly_report.html) used in creating the [Weekly RSC Communicable and Respiratory Disease Report for England](https://www.rcgp.org.uk/representing-you/research-at-rcgp/research-surveillance-centre/public-health-data) ("The Weekly Report") which is part of our work funded by the [UK Health Security Agency](https://www.gov.uk/government/organisations/uk-health-security-agency).
